# 儿童缺铁——最常见的营养缺口以及怎么补

**⏱️ 快速摘要（30秒）**

🟡 **紧急程度：重要——缺铁是全球儿童最常见的营养缺乏，而且可能以难以逆转的方式影响大脑发育。**

**简单说：** 铁不起眼，但不可或缺。孩子需要铁来制造血红蛋白（在血液中携带氧气）、支持大脑发育，以及维持免疫系统正常运转。缺铁的发生率高得惊人——它是全球儿童中最普遍的营养缺乏——而且在三个高风险时期打击最重：6–24个月、幼儿期（尤其是牛奶喝太多的孩子），以及青春期（尤其是月经来潮的女孩）。好消息是：注意饮食再加上必要时的补充，完全可以预防和治疗。

**核心要点：**
- 缺铁可能影响大脑发育——这不是可以忽略的 [来源：src3]
- 宝宝从6个月起需要富含铁的辅食，因为出生时的铁储备开始耗尽 [来源：src2]
- 近10%的1–3岁幼儿缺铁，很多时候是因为牛奶喝太多 [来源：src4]
- 月经来潮的女孩每天需要15毫克铁——远高于同龄男孩 [来源：src5]
- 含铁食物搭配维生素C能显著提高吸收率 [来源：src3]

---

## 铁为什么重要

铁在孩子体内承担几项关键任务：

- **运输氧气**——铁是血红蛋白的核心成分，血红蛋白负责把氧气从肺部输送到全身每个组织
- **支持大脑发育**——铁对神经递质合成、髓鞘化（神经纤维的绝缘层）和大脑的能量代谢至关重要 [来源：src3]
- **驱动免疫系统**——缺铁的孩子更容易生病
- **提供能量和支撑生长**——铁参与细胞层面的能量产生

铁不够时，身体会优先保证血液里的氧气输送，这意味着大脑、肌肉和免疫系统可能在贫血出现在化验单上之前就已经受到"静悄悄的亏欠"了。

### 大脑发育的隐忧

这是缺铁比听起来更严重的原因。在大脑发育的关键时期——特别是婴儿期和幼儿期——缺铁可能影响认知功能、行为和运动发育。[来源：src3] 一些研究提示，早期严重缺铁对大脑发育的影响即使在铁水平恢复后也可能无法完全逆转。

这不是为了制造恐慌——而是要引起重视。缺铁很常见，容易检测，也容易治疗。但需要及时发现和处理。

## 各年龄段的铁需求

| 年龄 | 每天铁需求 | 重点说明 |
|---|---|---|
| 0–6个月 | 母乳或配方奶可以满足 | 早产宝宝可能需要额外补充 |
| 7–12个月 | 11毫克 | 出生铁储备在6个月左右耗尽；富含铁的辅食至关重要 [来源：src2] |
| 1–3岁 | 7毫克 | 牛奶喝太多是这个年龄段缺铁的头号原因 [来源：src4] |
| 4–8岁 | 10毫克 | 不吃肉和豆类的挑食孩子有风险 |
| 9–13岁 | 8毫克 | 需求略低，但仍然重要 |
| 14–18岁（男孩） | 11毫克 | 生长突增的需求 |
| 14–18岁（女孩） | 15毫克 | 月经失血显著增加需求 [来源：src5] |

---

## 三个高风险窗口

### 窗口一：6–24个月——出生铁储备耗尽

宝宝出生时带有在孕晚期积累的铁储备。这些储备大约够用4–6个月。之后，宝宝需要从食物中（有时需要补充剂）获取铁，以满足快速生长的需求。[来源：src2]

与此同时，母乳虽然在几乎所有其他方面都很优秀，但铁含量相对较低。这就是为什么**富含铁的食物应该是最早添加的辅食之一**，大约在6个月左右。[来源：src3]

对于母乳喂养的宝宝，AAP建议从4个月起补充铁（每天1毫克/公斤体重），直到含铁辅食吃得比较稳定为止。[来源：src1]

**优质的富含铁的第一口食物：**
- 强化铁的婴儿米粉
- 泥状或切碎的肉类（牛肉、鸡肉、火鸡肉、羊肉）
- 捣碎的豆子或扁豆
- 嫩豆腐泥
- 深色叶菜泥

### 窗口二：幼儿期（1–3岁）——牛奶的问题

近**10%的1–3岁幼儿缺铁**，其中大约一半发展为缺铁性贫血。[来源：src4] 主要原因不是环境里缺少含铁食物——而是牛奶喝太多了。

机制是这样的：喝大量牛奶的幼儿，肚子被奶填满了，就吃不下含铁的食物。牛奶本身含铁极少。而且大量牛奶实际上会抑制铁吸收，并可能导致微量的肠道出血，造成额外的铁流失。[来源：src4]

**解决办法很简单：** 12个月以后，牛奶每天限在**480–720毫升（2–3杯）以内**。[来源：src4] 牛奶安排在正餐时喝，不要当全天饮料。确保每顿饭都有含铁的食物。

### 窗口三：青春期——月经和生长

缺铁是全世界最常见的营养缺乏，**青春期女孩是最高危的群体之一。**[来源：src5]

原因叠加：月经失血（平均每个周期30–40毫升）造成持续的铁流失；生长突增增加铁需求；很多青少年的饮食偏向加工食品、缺少含铁的完整食物。

月经来潮的青少年女孩每天推荐摄入量为**15毫克**——远高于同龄男孩的11毫克。[来源：src5]

任何年龄段的其他高风险人群还有：
- 早产或低出生体重的婴儿
- 有慢性疾病或饮食限制的孩子
- 素食和纯素食的孩子（植物性铁的生物利用率较低）
- 低收入家庭中食物获取有限的孩子
- 青少年运动员，尤其是女性运动员

## 含铁食物——实用指南

### 动物来源（血红素铁——吸收率最高）

| 食物 | 每份大约含铁量 |
|---|---|
| 牛肉（85克） | 2.5毫克 |
| 鸡腿肉（85克） | 1.1毫克 |
| 火鸡肉（85克） | 1.4毫克 |
| 猪肉（85克） | 0.9毫克 |
| 三文鱼（85克） | 0.3毫克 |
| 鸡蛋（1个） | 0.9毫克 |

### 植物来源（非血红素铁——吸收率较低但仍有价值）

| 食物 | 每份大约含铁量 |
|---|---|
| 强化铁的谷物（1杯） | 4–18毫克（差异很大——看标签） |
| 扁豆（半杯煮熟） | 3.3毫克 |
| 黑豆（半杯煮熟） | 1.8毫克 |
| 豆腐（半杯） | 3.4毫克 |
| 菠菜（半杯煮熟） | 3.2毫克 |
| 强化燕麦粥（1小包） | 4–8毫克 |

### 维生素C的增效技巧

非血红素铁（来自植物和强化谷物）如果和维生素C一起吃，吸收效果会好得多。[来源：src3] 这是最简单也最有效的营养策略之一：

- 强化铁谷物配草莓或柑橘
- 扁豆或豆子上挤点柠檬汁
- 豆类菜肴配西兰花或甜椒
- 含铁丰富的正餐配番茄酱

### 什么会抑制铁吸收

有些物质在同一餐中会降低铁的吸收：
- **钙（乳制品）**——含铁丰富的正餐不要把牛奶当主要饮料
- **单宁（茶、咖啡）**——对小孩子影响不大，但对青少年有关系
- **植酸（全谷物、豆类）**——浸泡、发芽或发酵可以降低植酸含量

这不意味着要回避这些食物——只是搭配上讲点策略。比如把牛奶安排在两餐之间喝，而不是配着含铁丰富的晚餐一起喝。

## 缺铁的表现

缺铁是逐步发展的，早期可能没有明显症状。随着加重：

**早期表现：**
- 容易疲倦、精力不足
- 皮肤苍白（查看下眼睑内侧、指甲床和手掌——这些比整体肤色更可靠，对不同肤色的孩子都适用）
- 食欲下降
- 易烦躁

**更严重的表现：**
- 乏力
- 注意力不集中
- 容易生病
- 心跳加快，特别是运动时
- 怕冷
- 指甲变脆
- 幼儿：运动或语言发育迟缓

**一个值得注意的信号：** 渴望或吃非食物的东西（冰块、泥土、粉笔、纸张）——叫做**异食癖**——可能是缺铁的信号。如果孩子在吃或渴望非食物的东西，跟儿科医生提一下。

### 什么时候该筛查

AAP建议在大约12个月时做一次缺铁筛查血检。[来源：src1] 以下情况可能需要额外筛查：
- 早产或低出生体重的宝宝（更早筛查）
- 任何年龄有风险因素的孩子
- 月经来潮的青少年，尤其是月经量大的
- 有缺铁症状的孩子

筛查是简单的抽血检查，测血红蛋白，最好同时测血清铁蛋白（反映铁储备，能在贫血出现之前就发现缺铁）。

## 治疗

如果孩子被确诊为缺铁，治疗取决于严重程度：

**轻度缺铁（铁蛋白低，血红蛋白正常）：**
- 调整饮食——增加含铁食物、减少过量的牛奶、搭配维生素C
- 可能仅靠饮食调整就能纠正

**缺铁性贫血（血红蛋白低）：**
- 儿科医生开具铁补充剂——通常每天每公斤体重3–6毫克的元素铁
- 空腹或搭配维生素C服用，吸收最好
- 精力和食欲通常在1–2周内改善；血常规在1–2个月内恢复正常
- 血常规恢复后需要继续补充2–3个月以补满储备

**铁补充剂的副作用：**
- 便秘（最常见）
- 大便颜色变深（正常无害）
- 胃不舒服或恶心

如果便秘是问题，儿科医生可能调整剂量、换一种配方，或建议缓解便秘的方法。铁补充剂要严格按医嘱服用——摄入过多铁是有毒的。

**重要安全提醒：** 把铁补充剂放在孩子够不到的地方。铁补充剂过量是幼儿中毒致死的主要原因之一。妥善保管、小心取用。

## 什么时候该找医生

**🔴 尽快联系医生：**
- 孩子出现严重贫血的迹象——极度疲倦、心跳快、活动时呼吸困难、头晕或晕倒
- 异食癖——孩子在吃非食物的东西（泥土、冰块、粉笔、纸张）
- 幼儿运动或语言发育似乎落后——缺铁可能是一个原因

**🟡 下次儿保体检时提一提：**
- 孩子挑食，大部分含铁食物都不吃
- 幼儿每天牛奶喝超过720毫升
- 青少年月经量大且看起来疲倦
- 担心素食或纯素食饮食的铁摄入
- 孩子是早产儿，想讨论铁补充

> **换个角度看：** 缺铁非常常见，表现又不那么明显。大多数家长注意不到，因为那些症状——疲倦、烦躁、食欲差——跟正常的童年生活太像了。这正是为什么筛查很重要，也是为什么从6个月起有意识地保证含铁食物是回报最高的营养策略之一。不需要每顿饭都数毫克。只需要让含铁食物稳定地出现在餐桌上，同时管好牛奶的量。

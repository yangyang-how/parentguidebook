# 预防窒息和紧急处理——家长必读指南

**⏱️ 快速摘要（30秒）**

🔴 **紧急程度：危急——窒息可以在几分钟内危及生命。在需要用到这些知识之前，先学会。**

**你需要知道的：**
- 异物吸入是三岁以下儿童意外死亡的第四大原因，在一岁以下婴儿中是第三大原因 [来源：src1]
- 最高风险期是6个月到3岁，这时孩子开始吃固体食物，什么都往嘴里放 [来源：src1]
- 最危险的窒息食物是又圆、又硬、又滑的：热狗、整颗葡萄、坚果、硬糖、爆米花和生胡萝卜 [来源：src1]
- 急救方法因年龄而异：一岁以下用拍背法和胸部按压；一岁以上用腹部冲击法（海姆立克法）[来源：src4]
- 幼儿的食物应切成不超过1厘米的小块，圆形食物必须纵向切成四瓣 [来源：src2]
- 每位家长和看护人都应该学习心肺复苏（CPR）和窒息急救课程 [来源：src2]

**如果孩子正在窒息：** 直接跳到下面的[紧急处理](#紧急处理按年龄分)部分。

## 幼儿为什么容易窒息

三岁以下的孩子窒息风险最高，原因完全是发育阶段决定的——不是谁做错了什么。

**解剖结构不利。** 幼儿的气道很窄。成人能轻松咳出来的一块食物，可能完全堵住学步期孩子的气管。一岁孩子的气道直径大约相当于一根吸管粗细——不需要多大的东西就能堵住。

**还在学咀嚼。** 宝宝和学步期的孩子还没有长全臼齿，不能像大孩子和成人那样研磨食物。可能咬下一块塞得进嘴但嚼不够碎就吞下去了。[来源：src1]

**用嘴探索世界。** 6个月到3岁之间，孩子几乎什么东西都往嘴里放。这是正常且重要的发育过程——但这意味着硬币、纽扣电池、小玩具零件、气球和其他非食物物品也是窒息隐患。[来源：src3]

**吃东西时注意力分散。** 幼儿会边走边吃、边跑边吃、边笑边吃、边哭边吃、边玩边吃。这些行为都会增加食物呛入气道的风险。躺着或半躺着吃东西同样增加风险。

## 最危险的窒息物品

### 高风险食物

并非所有食物的危险程度相同。最高风险的食物有几个共同特征：圆的（能封住气道）、硬的（咳不碎）、滑的（难以抓住取出）。[来源：src1]

| 食物 | 为什么危险 | 怎样处理更安全 |
|------|-----------|---------------|
| 热狗 | 又圆又硬，大小刚好能堵住孩子气道 | 先纵切成四条，再切成小块 |
| 整颗葡萄 | 圆而且滑——直接滑进气道 | 纵向切成四瓣（只切一半还是圆的，不够安全） |
| 坚果和种子 | 硬、形状不规则，没有臼齿嚼不碎 | 四岁以下完全不给；可以用坚果酱（薄薄一层，不要一大团） |
| 爆米花 | 形状不规则，有硬的玉米壳碎片 | 四岁以下完全不给 |
| 硬糖和润喉糖 | 圆、滑、溶解慢 | 四岁以下完全不给 |
| 生胡萝卜 | 硬，咬碎后会断成尖锐碎片 | 煮软后切成细条 |
| 大块奶酪 | 硬且有弹性——可以变形贴合气道 | 切成薄片或小丁 |
| 大块肉 | 韧，嚼不够碎 | 撕成丝或切成很小的块 |
| 花生酱（大团的） | 黏稠——难以从气道清除 | 在面包或饼干上薄薄涂一层；绝对不要用勺子直接喂 |
| 棉花糖 | 有弹性——会变形贴合气道 | 幼儿不要吃 |

### 高风险非食物物品

- **气球**——乳胶气球对八岁以下儿童都有窒息风险，在非食物窒息死亡中占比格外高 [来源：src2]
- **硬币**——最常被误吞的非食物物品之一 [来源：src3]
- **纽扣电池**——既是窒息风险也是化学灼伤风险；一旦吞入必须立即急诊处理
- **小玩具零件**——能穿过卫生纸筒的东西，对三岁以下的孩子来说就太小了
- **磁铁**——小而强力的磁铁（比如桌面玩具或大孩子的拼搭套件里的磁铁）如果吞入两颗以上，会在体内互相吸引，造成严重内伤
- **笔帽和记号笔帽**

## 怎样安全地切食物

这是最简单、最有效的预防措施之一。AAP建议给宝宝和幼儿的食物切成不超过半英寸（约1厘米）的小块。[来源：src2]

**几条关键规则：**

1. **圆形食物先纵向切成四瓣，再横切成小块。** 葡萄只切一半还是圆的——仍然可能封住气道。纵切成四瓣，每一瓣都有平面，不会形成密封。

2. **热狗：同样的规则。** 先纵切成四条，再切小块。绝对不要给孩子一个圆形的热狗切片。

3. **硬蔬菜煮到软再给。** 生胡萝卜、芹菜和苹果片应该蒸煮到轻压就软的程度，再给幼儿吃。

4. **难嚼的肉类撕碎或切细。** 肉和禽类应该切成很小的块或撕成丝，不要给大块。

5. **坚果酱薄薄涂。** 一大团花生酱可以堵住气道。薄薄涂在面包、饼干或香蕉上。

6. **去掉所有果核、种子和骨头**再给孩子吃。

## 减少风险的用餐和看护习惯

预防远比紧急处理有效。以下习惯能显著降低窒息风险：

- **吃饭时一定要有人看着。** 四岁以下的孩子任何时候吃东西都不能无人看管。这是最重要的预防措施。
- **孩子吃东西时应该坐直**，坐在餐椅或桌前——绝不能边走边吃、边跑边吃、躺着吃或在车里吃。
- **不要在大笑或大哭时吃东西。** 如果孩子正在闹情绪或笑得停不下来，等平静下来再吃。
- **教孩子好好嚼。** 鼓励孩子嚼碎再咽。自己做示范——夸张地咀嚼对正在学吃饭的学步期孩子很有帮助。
- **给适合年龄的食物。** 不要不做调整就把给5岁孩子吃的食物直接给1岁的孩子。
- **保持地面没有小物件。** 如果家里有大孩子玩小玩具（乐高积木、弹珠、手工珠子），把这些东西放在小孩子够不到的单独房间或高处。
- **检查家具底下和沙发垫缝隙。** 孩子会在大人想不到的地方找到东西。

## 紧急处理（按年龄分）

**本节是参考资料，不能替代实操培训。请参加CPR课程。** 看文字描述和在训练假人上练习是两回事。AAP建议所有家长和看护人完成婴幼儿CPR和窒息急救培训。[来源：src2]

### 第一步：孩子真的窒息了吗？

如果孩子在**用力咳嗽**，还没有到窒息的程度——身体在做正确的事。在孩子有效咳嗽时不要拍背或做腹部冲击。保持冷静，留在身边，鼓励继续咳，随时准备在咳嗽变弱或无声时采取行动。

**严重（完全）气道阻塞的表现：**
- 不能咳嗽、不能哭、发不出声音
- 嘴在动——想呼吸但没有气流
- 皮肤变蓝变灰，特别是嘴唇和面部
- 失去意识

**这时就要立刻行动。**

### 一岁以下婴儿：拍背法和胸部按压

[来源：src4]

**不要对一岁以下的婴儿使用腹部冲击法（海姆立克法）。** 婴儿的内脏还太脆弱。

1. **让婴儿面朝下趴在你的前臂上**，头部低于胸部。用手托住头部和下颌。前臂搁在大腿上保持稳定。
2. **用另一只手的掌根在两肩胛骨之间用力拍5下。** 每一下都是一个明确有力的击打——目的是在异物后方产生足够的压力把它推出来。
3. **翻转婴儿使其面朝上**趴在另一只前臂上，头部仍低于胸部。
4. **用两根手指在胸骨上、乳头连线稍下方按压5下。** 垂直向下按大约4厘米，然后松开。每次按压都应该是一个独立、有力的动作。
5. **交替进行：5下拍背，5下胸部按压。** 持续进行直到异物排出、婴儿开始咳嗽或哭泣、或婴儿失去意识。
6. **如果婴儿失去意识：** 开始婴儿CPR。如果还没有打急救电话，立即拨打。每次打开气道准备吹气时，看一眼口腔——如果看到异物就取出来。不要盲目用手指掏。

### 一岁以上儿童：腹部冲击法（海姆立克法）

[来源：src4]

1. **站在或跪在孩子身后。** 对于较小的孩子，跪下来可能更合适。
2. **一只手握拳。** 把拳头的拇指侧贴在孩子腹部，肚脐上方、胸骨下方的位置。
3. **另一只手握住拳头。**
4. **快速向内向上冲击**——像要用拳头把孩子从地面提起来一样。每次冲击都是一个明确的动作。
5. **重复**直到异物排出或孩子失去意识。
6. **如果孩子失去意识：** 把孩子放平在地上，拨打急救电话，开始CPR。每次打开气道准备吹气时，看一眼口腔——如果看到异物就取出来。不要盲目用手指掏。

### 拨打急救电话

- **如果孩子没有呼吸或失去意识**——立即拨打急救电话（或让别人打，你同时开始急救）
- **如果只有你一个人**——先做2分钟拍背/胸部按压或腹部冲击，然后打急救电话，再继续急救
- **任何需要急救干预的窒息事件之后**——即使异物出来了、孩子看起来完全没事，也要带孩子去看医生。异物可能部分脱落后滑入更深的气道，组织也可能受损。

## 窒息事件之后

即使异物出来了、孩子看起来完全没事，也应该去看医生。原因如下：

- **部分吸入：** 异物的一部分可能碎裂后进入下呼吸道，在接下来几天引起感染或炎症。窒息事件后突然出现的持续咳嗽、喘息或发烧，应该立即就医。[来源：src3]
- **气道损伤：** 用力咳嗽、拍背或腹部冲击可能造成瘀伤或轻微损伤，需要观察。
- **延迟症状：** 有些异物卡在气道里不会立即引起不适，但数小时或数天后才出现症状——同一侧肺部反复出现肺炎是典型的未发现气道异物的征兆。[来源：src3]

如果吸入的是硬物（硬币、玩具零件、电池），可能需要通过硬式支气管镜取出。这是儿科专科医生常规操作的标准手术。[来源：src3]

## 什么时候拨打急救电话

🔴 **出现以下情况立即拨打急救电话：**

- 孩子不能呼吸、不能咳嗽、发不出声音
- 孩子嘴唇或面部变蓝变灰
- 窒息过程中孩子失去意识
- 窒息事件后孩子仍然呼吸困难，即使异物似乎已经出来
- 怀疑孩子吞了纽扣电池（同时也是中毒急症）

🟡 **当天应该看医生的情况：**

- 孩子窒息后自行缓解，但现在持续咳嗽或声音嘶哑
- 你不确定异物是否完全排出
- 孩子在把东西放进嘴里之后出现喘息或呼吸困难——即使你没有看到窒息过程

## 去学CPR

这篇文章给你知识。CPR课程给你肌肉记忆。

孩子窒息的时候，你没有时间看说明书。你不会思路清晰。你需要急救步骤刻在手上，而不是存在屏幕里。AAP建议所有家长、祖父母和经常照看孩子的人完成婴幼儿CPR和窒息急救培训。[来源：src2]

**哪里学：**
- 美国红十字会和美国心脏协会都提供婴幼儿CPR课程，有线下班也有线上班（线下实操课效果明显更好）
- 很多医院、社区中心和消防站提供免费或低价课程
- 可以问孩子的儿科医生——他们可能知道当地资源
- 有些单位和宗教团体会组织团体培训

如果费用或交通是障碍，美国心脏协会有免费在线视频课程。线下培训最好，但看高质量视频并在枕头上练习动作，也比什么都不会强得多。

**每两年复习一次。** 技术会更新，肌肉记忆也会退化。

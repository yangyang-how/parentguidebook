# 6个月至2岁的呼吸问题——喉炎、窒息与气道急症

**⏱️ 快速摘要（30秒）**

🟡 **紧急程度：重要——喉炎和窒息在这个年龄段很常见，知道怎么应对至关重要。**

**简单说：** 6个月到2岁之间，孩子开始吃辅食、爬行、走路，什么东西都往嘴里塞。这带来两大呼吸风险：**喉炎**（一种病毒感染，犬吠样咳嗽听着吓人但通常是轻症）和**食物或小物件窒息**（3岁以下儿童意外死亡的第四大原因）。第二个RSV（呼吸道合胞病毒）流行季仍可能发生毛细支气管炎。家长需要学会辨别喘鸣音、掌握不同年龄段的海姆立克急救法，并知道什么情况该去急诊。

**关键信息：**
- 喉炎影响6个月至3岁儿童中约3%的孩子；94%的急诊病例当天即可出院 [来源：src1]
- 窒息是3岁以下儿童意外死亡的第四大原因 [来源：src5]
- 1岁以下用拍背+胸部冲击法；1岁以上用腹部冲击法（海姆立克法）[来源：src8]
- 6岁以下儿童不应使用止咳药 [来源：src2]

## 6个月至2岁的正常呼吸

到6个月时，呼吸应该比新生儿期更加规律和安静。这个年龄段的正常呼吸频率约为每分钟20到30次。间歇性呼吸应该已经消失。呼吸看起来应该毫不费力——正常呼吸时不应该看到肋间皮肤凹陷或鼻翼扇动。

孩子的免疫系统正在第一次接触各种病毒，呼吸道感染很常见——尤其是开始上托班之后。大多数是轻微的上呼吸道感染（普通感冒），会自行好转。关键是分清哪些症状只是感冒，哪些需要就医。

## 喉炎：犬吠样咳嗽

如果你从没听过喉炎的咳嗽声，第一次听到一定会被吓到——那是一种粗糙、像海豹叫一样的犬吠样咳嗽，非常有辨识度。孩子还可能出现**喘鸣**：吸气时发出高调、沙哑的声音。这两种症状都是因为喉部和气管肿胀所致。[来源：src10]

### 病因

副流感病毒约占所有喉炎病例的75%。这是病毒感染——抗生素没有用。[来源：src1]

### 典型病程

喉炎通常先出现感冒症状（流鼻涕、轻微咳嗽），持续一两天后犬吠样咳嗽开始出现——**通常在夜间加重**。这就是为什么很多家长第一次遇到喉炎，都是凌晨两点被吓醒。[来源：src10]

大多数病例属于轻症，3到5天即可好转。不到6%的急诊喉炎患儿需要住院。[来源：src1, src3]

### 处理方法

**居家护理（轻度喉炎——有犬吠样咳嗽但安静时无喘鸣）：**
- **保持镇定。** 哭闹和烦躁会加重喉炎症状。安抚孩子、让孩子放松。[来源：src2]
- **凉爽的夜间空气**可以暂时缓解症状。经典的"开车送急诊"路上症状常常好转，就是因为吸入了凉空气——很多家长到医院时，孩子已经明显改善了。[来源：src3]
- **房间里放一台冷雾加湿器**可能有帮助，但证据有限。[来源：src2]
- **让孩子保持坐姿**——坐着比平躺更舒服。

**不要用的药物：** 6岁以下儿童不应使用止咳药。病毒性喉炎用抗生素无效。[来源：src2]

**医疗治疗：**
- **地塞米松**（一种糖皮质激素），单次口服即高度有效，是一线治疗方案。它能减轻气道肿胀，通常数小时内即可改善症状。[来源：src1]
- **雾化肾上腺素**用于急诊中的中重度喉炎，起效快但作用时间短。接受雾化治疗的孩子需要观察2到4小时，因为症状可能反弹。[来源：src3]

### 喉炎何时变成急症

**🔴 出现以下情况立即去急诊：**
- **安静时出现喘鸣**——如果孩子在安静、不哭闹时仍能听到高调的吸气声，说明气道已经明显狭窄 [来源：src3]
- **流口水或吞咽困难**——可能提示会厌炎（Hib疫苗普及后已少见，但仍然严重）
- **嘴唇或面部发青**
- **严重凹陷**——每次呼吸时皮肤深深地往肋骨间凹进去
- **孩子看起来很害怕，一动不动地坐着、身体前倾**——这种姿势提示气道严重受阻

## 喘鸣与哮鸣：分清两种声音

这两种声音来自气道的不同部位，能准确区分有助于和医生沟通：[来源：src4]

| | 喘鸣（stridor） | 哮鸣（wheezing） |
|---|---|---|
| **声音特征** | 高调、沙哑 | 高调、哨笛样 |
| **出现时机** | 吸气时 | 呼气时 |
| **病变部位** | 上气道（咽喉、喉部） | 下气道（肺部、细支气管） |
| **常见原因** | 喉炎、异物吸入 | 毛细支气管炎、哮喘 |

**为什么要分清：** 打电话给医生或到急诊时说"孩子有喘鸣"，医生立刻知道问题在上气道。这是一条具体、有用的信息。[来源：src4]

## 窒息与异物吸入

孩子开始吃辅食（大约6个月起）、开始爬行、用嘴巴探索世界，窒息就成了切实的风险。异物吸入是3岁以下儿童意外死亡的第四大原因，也是1岁以下婴儿意外死亡的第三大原因。[来源：src5]

### 高风险食物

对幼儿最危险的食物是圆形、硬质或黏性的：
- **热狗**——儿童窒息食物排名第一。纵向切成条状，再切成小块。[来源：src5]
- **葡萄**——纵向切成四瓣 [来源：src6]
- **坚果和种子**——4岁前不要给 [来源：src5]
- **爆米花**——4岁前不要给 [来源：src5]
- **硬糖** [来源：src5]
- **生胡萝卜**——煮软后再切成合适大小 [来源：src5]
- **大块肉或奶酪**——切成不超过1厘米的小块 [来源：src6]
- **花生酱**——不要给一大团；薄薄地抹在面包或饼干上 [来源：src6]

### 食物处理原则

- 所有食物切成不超过1厘米的小块 [来源：src6]
- 圆形食物（葡萄、小番茄、热狗）纵向切成四瓣 [来源：src6]
- 硬的水果和蔬菜煮软再给 [来源：src6]
- 吃东西时必须有人看着——不能边走边吃、边跑边吃、边玩边吃 [来源：src6]

### 非食物窒息隐患

- **气球**——8岁以下都有窒息风险（未充气的乳胶气球和破碎的气球碎片）[来源：src6]
- **硬币**——最常被误吸的非食物物品之一 [来源：src7]
- **小零件玩具**——严格遵守包装上的年龄建议 [来源：src7]
- **纽扣电池**——吞入后极其危险（不仅有窒息风险，还会造成化学灼伤）

### 孩子窒息时怎么办

**1岁以下婴儿：** [来源：src8]
1. 将宝宝面朝下放在前臂上，托住头部
2. 用手掌根部在肩胛骨之间用力拍背5次
3. 翻转宝宝面朝上，用两根手指在胸骨上按压5次
4. 交替拍背和胸部按压，直到异物排出或宝宝开始咳嗽/哭出声
5. 如果宝宝失去意识，立即拨打120并开始婴儿心肺复苏

**1岁以上儿童：** [来源：src8]
1. 站在孩子身后，双臂环抱腰部
2. 握拳放在肚脐上方、肋弓下方
3. 快速向上冲击（海姆立克法）
4. 持续操作直到异物排出或孩子失去意识
5. 如果失去意识，立即拨打120并开始儿童心肺复苏

> **每位家长和看护者都应该学会婴儿/儿童心肺复苏。** 美国儿科学会（AAP）建议所有家长接受心肺复苏培训。当地医院、红十字会和社区中心通常都有培训课程。只需要几个小时，关键时刻可以救命。[来源：src6]

## RSV/毛细支气管炎的持续风险

孩子在第一个和第二个RSV流行季都可能发生毛细支气管炎。第二季的感染通常比第一次轻。治疗仍以对症支持为主：补充水分、鼻腔吸引和密切观察。[来源：src9]

如果孩子尚未接种尼塞韦单抗（nirsevimab）且即将进入第一个RSV流行季，请咨询儿科医生是否仍有必要接种。

## 何时拨打急救电话

**出现以下情况拨打120或立即去急诊：**
- 安静时出现喘鸣（不仅仅是哭闹时）[来源：src3]
- 嘴唇或面部发青 [来源：src4]
- 流口水且无法吞咽 [来源：src3]
- 严重凹陷——肋间或肋下皮肤随呼吸深深凹陷 [来源：src4]
- 突发窒息——如果孩子还能用力咳嗽，让孩子继续咳。如果无法咳嗽、无法哭出声、无法呼吸，立即对1岁以下婴儿执行拍背/胸部按压，对1岁以上儿童执行海姆立克法，同时拨打120。[来源：src8]
- 孩子异常软弱无力、嗜睡或失去反应
